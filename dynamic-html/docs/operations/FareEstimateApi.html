This is the API

fareEstimateGetFareEstimateByZone

URL

https://timetableapi.ptv.vic.gov.au//v3/fare_estimate/min_zone/{minZone}/max_zone/{maxZone}

HTTP Method

GET

Response Type

V3.FareEstimateResponse

Parameters

  • Path: minZone Integer

    Minimum Zone travelled through ie. 1

  • Path: maxZone Integer

    Maximum Zone travelled through id. 6

  • Query: journey_touch_on_utc Date

    JourneyTouchOnUtc in format yyyy-M-d h:m (e.g 2016-5-31 16:53).

  • Query: journey_touch_off_utc Date

    JourneyTouchOffUtc in format yyyy-M-d h:m (e.g 2016-5-31 16:53).

  • Query: is_journey_in_free_tram_zone Boolean

  • Query: travelled_route_types List(Integer)

  • Query: token String

    Please ignore

  • Query: devid String

    Your developer id

  • Query: signature String

    Authentication signature for request
