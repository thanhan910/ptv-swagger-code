disruptionsGetAllDisruptions

URL

https://timetableapi.ptv.vic.gov.au//v3/disruptions

HTTP Method

GET

Response Type

V3.DisruptionsResponse

Parameters

  • Query: route_types List(Integer)

    Filter by route_type; values returned via RouteTypes API

  • Query: disruption_modes List(Integer)

    Filter by disruption_mode; values returned via v3/disruptions/modes API

  • Query: disruption_status String

    Filter by status of disruption

  • Query: token String

    Please ignore

  • Query: devid String

    Your developer id

  • Query: signature String

    Authentication signature for request
