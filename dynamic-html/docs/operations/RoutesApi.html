This is the API

routesOneOrMoreRoutes

URL

https://timetableapi.ptv.vic.gov.au//v3/routes

HTTP Method

GET

Response Type

V3.RouteResponse

Parameters

  • Query: route_types List(Integer)

    Filter by route_type; values returned via RouteTypes API

  • Query: route_name String

    Filter by name of route (accepts partial route name matches)

  • Query: token String

    Please ignore

  • Query: devid String

    Your developer id

  • Query: signature String

    Authentication signature for request

routesRouteFromId

URL

https://timetableapi.ptv.vic.gov.au//v3/routes/{route_id}

HTTP Method

GET

Response Type

V3.RouteResponse

Parameters

  • Path: route_id Integer

    Identifier of route; values returned by Departures, Directions and Disruptions APIs

  • Query: include_geopath Boolean

    Indicates kif geopath data will be returned (default = false)

  • Query: geopath_utc Date

    Filter geopaths by date (ISO 8601 UTC format) (default = current date)

  • Query: token String

    Please ignore

  • Query: devid String

    Your developer id

  • Query: signature String

    Authentication signature for request
