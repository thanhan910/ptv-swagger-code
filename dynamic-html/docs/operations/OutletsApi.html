outletsGetAllOutlets

URL

https://timetableapi.ptv.vic.gov.au//v3/outlets

HTTP Method

GET

Response Type

V3.OutletResponse

Parameters

  • Query: max_results Integer

    Maximum number of results returned (default = 30)

  • Query: token String

    Please ignore

  • Query: devid String

    Your developer id

    Authentication signature for request

outletsGetOutletsByGeolocation

URL

https://timetableapi.ptv.vic.gov.au//v3/outlets/location/{latitude},{longitude}

HTTP Method

GET

Response Type

V3.OutletGeolocationResponse

Parameters

  • Path: latitude Float

    Geographic coordinate of latitude

  • Path: longitude Float

    Geographic coordinate of longitude

  • Query: max_distance Double

    Filter by maximum distance (in metres) from location specified via latitude and longitude parameters (default = 300)

  • Query: max_results Integer

    Maximum number of results returned (default = 30)

  • Query: token String

    Please ignore

  • Query: devid String

    Your developer id

  • Query: signature String

    Authentication signature for request
