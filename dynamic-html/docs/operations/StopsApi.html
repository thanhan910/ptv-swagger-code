This is the API

stopsStopDetails

URL

https://timetableapi.ptv.vic.gov.au//v3/stops/{stop_id}/route_type/{route_type}

HTTP Method

GET

Response Type

V3.StopResponse

Parameters

  • Path: stop_id Integer

    Identifier of stop; values returned by Stops API

  • Path: route_type Integer

    Number identifying transport mode; values returned via RouteTypes API

  • Query: stop_location Boolean

    Indicates if stop location information will be returned (default = false)

  • Query: stop_amenities Boolean

    Indicates if stop amenity information will be returned (default = false)

  • Query: stop_accessibility Boolean

    Indicates if stop accessibility information will be returned (default = false)

  • Query: stop_contact Boolean

    Indicates if stop contact information will be returned (default = false)

  • Query: stop_ticket Boolean

    Indicates if stop ticket information will be returned (default = false)

  • Query: gtfs Boolean

    Incdicates whether the stop_id is a GTFS ID or not

  • Query: stop_staffing Boolean

    Indicates if stop staffing information will be returned (default = false)

  • Query: stop_disruptions Boolean

    Indicates if stop disruption information will be returned (default = false)

  • Query: token String

    Please ignore

  • Query: devid String

    Your developer id

  • Query: signature String

    Authentication signature for request

stopsStopsByGeolocation

URL

https://timetableapi.ptv.vic.gov.au//v3/stops/location/{latitude},{longitude}

HTTP Method

GET

Response Type

V3.StopsByDistanceResponse

Parameters

  • Path: latitude Float

    Geographic coordinate of latitude

  • Path: longitude Float

    Geographic coordinate of longitude

  • Query: route_types List(Integer)

    Filter by route_type; values returned via RouteTypes API

  • Query: max_results Integer

    Maximum number of results returned (default = 30)

  • Query: max_distance Double

    Filter by maximum distance (in metres) from location specified via latitude and longitude parameters (default = 300)

  • Query: stop_disruptions Boolean

    Indicates if stop disruption information will be returned (default = false)

  • Query: token String

    Please ignore

  • Query: devid String

    Your developer id

  • Query: signature String

    Authentication signature for request

stopsStopsForRoute

URL

https://timetableapi.ptv.vic.gov.au//v3/stops/route/{route_id}/route_type/{route_type}

HTTP Method

GET

Response Type

V3.StopsOnRouteResponse

Parameters

  • Path: route_id Integer

    Identifier of route; values returned by Routes API - v3/routes

  • Path: route_type Integer

    Number identifying transport mode; values returned via RouteTypes API

  • Query: direction_id Integer

    An optional direction; values returned by Directions API. When this is set, stop sequence information is returned in the response.

  • Query: stop_disruptions Boolean

    Indicates if stop disruption information will be returned (default = false)

  • Query: include_geopath Boolean

    Indicates if geopath data will be returned (default = false)

  • Query: geopath_utc Date

    Filter geopaths by date (ISO 8601 UTC format) (default = current date)

  • Query: token String

    Please ignore

  • Query: devid String

    Your developer id

  • Query: signature String

    Authentication signature for request
