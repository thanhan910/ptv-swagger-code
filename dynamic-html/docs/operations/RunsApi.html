This is the API

runsForRoute

URL

https://timetableapi.ptv.vic.gov.au//v3/runs/route/{route_id}

HTTP Method

GET

Response Type

V3.RunsResponse

Parameters

  • Path: route_id Integer

    Identifier of route; values returned by Routes API - v3/routes.

  • Query: expand List(String)

    List of objects to be returned in full (i.e. expanded) - options include: All, VehiclePosition, VehicleDescriptor, or None. Default is None.

  • Query: date_utc Date

    Date of the request. (optional - defaults to now)

  • Query: token String

    Please ignore

  • Query: devid String

    Your developer id

  • Query: signature String

    Authentication signature for request

runsForRun

URL

https://timetableapi.ptv.vic.gov.au//v3/runs/{run_ref}

HTTP Method

GET

Response Type

V3.RunsResponse

Parameters

  • Path: run_ref String

    The run_ref is the identifier of a run as returned by the departures/* and runs/* endpoints. WARNING, run_id is deprecated. Use run_ref instead.

  • Query: expand List(String)

    List of objects to be returned in full (i.e. expanded) - options include: All, VehiclePosition, VehicleDescriptor, or None. Default is None.

  • Query: date_utc Date

    Date of the request. (optional - defaults to now)

  • Query: include_geopath Boolean

    Indicates if geopath data will be returned (default = false)

  • Query: token String

    Please ignore

  • Query: devid String

    Your developer id

  • Query: signature String

    Authentication signature for request
